==========================================================================
Higgsfield AI API 결제 등록 및 과금 정책 요약
==========================================================================

Higgsfield AI는 크레딧(Credit) 시스템 기반으로 이미지 및 비디오 생성을 처리하는 
유료 구독형 크리에이티브 플랫폼입니다. API 연동을 위한 결제 정책은 다음과 같습니다.

1. 무료 체험 (Free Trial)
--------------------------------------------------------------------------
- 처음 회원가입 시 평가용 무료 크레딧이 일부 제공됩니다.
- 신용카드 결제 수단을 바로 등록하지 않고도 웹 대시보드 상에서 기본 생성을 
  테스트해볼 수 있습니다.
- 단, 무료 등급은 다음과 같은 제약이 있습니다:
  * 생성된 결과물에 워터마크 노출
  * 하루/한달 간 생성 가능한 횟수 제한
  * 고화질 및 최신 고성능 모델(Veo 3.1, Kling 3.0 등) 사용 불가
  * programmatic API 키 발급 및 호출 제약

2. 유료 구독 및 결제 수단 등록 (API 사용 시 필수)
--------------------------------------------------------------------------
- 개발자 콘솔에서 API Key(Key ID & Secret)를 활성화하여 외부 연동(Node.js, 
  Python 등)을 수행하려면 유료 구독 플랜(Starter, Creator, Business 등) 중 
  하나는 가입해야 합니다.
- 유료 등급 구독 및 추가 크레딧 구매를 위해 Higgsfield 플랫폼 내 결제 수단
  (신용카드 등, Stripe를 통한 보안 결제) 등록이 필수적입니다.
- 매월 할당된 크레딧이 소진되면 초과 이용을 위해 크레딧 충전이 필요합니다.
- 모델의 복잡도, 출력 해상도, 비디오 재생 시간(초)에 따라 크레딧 차감량이 
  유동적으로 적용됩니다. (예: 1080p, 캐릭터 유지 Soul ID 등은 추가 크레딧 소모)

3. API 키나 크레딧 없이 무료 테스트하는 방법 (Mock 모드)
--------------------------------------------------------------------------
본 프로젝트에 적용된 [Higgsfield API Playground (higgsfield.html)] 페이지는
인증 절차나 비용 지불 없이 로직을 검증할 수 있는 시뮬레이션을 탑재했습니다.

- 사용 방법:
  1. higgsfield.html 페이지의 'API 연결 설정'에서 '시뮬레이션 (Mock)' 선택
  2. 파라미터와 프롬프트를 입력하고 '생성 요청 시작' 클릭
- 장점:
  * 별도의 API Key나 결제 카드 등록이 없어도 서비스 전체 흐름 완벽 체험 가능
  * 대기열(queued) -> 렌더링(in_progress) -> 생성완료(completed) 상태 변화 로깅
  * 실제 API 통신 본문인 Request/Response JSON 데이터의 실시간 구조 검사 가능
  * 사전 정의된 고품질 데모 결과물 출력 및 갤러리 보관 가능
