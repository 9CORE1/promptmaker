Higgsfield v2 Client API 가이드

                            
이 문서는 애플리케이션 백엔드(Node.js/TypeScript)에서 Higgsfield SDK를 사용하여 이미지 및 비디오 생성을 통합할 때 필요한 표준 방법과 코드를 안내합니다.


                            

                                
                                보안 유의사항: Higgsfield v2 클라이언트는 브라우저 환경에서 직접 실행하는 것을 보안 상 금지하고 있습니다. API 호출은 서버측(Node.js, Express, Next.js 등)에서 수행해야 합니다.
                            


                            
1. 설치 (Installation)

                            
npm install @higgsfield/client


                            
2. 클라이언트 초기화 및 설정 (Configuration)

                            
인증 키(Key ID:Secret)를 사용하여 클라이언트를 설정합니다. 환경변수 HF_CREDENTIALS로 자동 주입할 수도 있습니다.

                            
import { config, higgsfield } from '@higgsfield/client/v2';

// 1. 명시적 인증 키 주입
config({ credentials: 'YOUR_KEY_ID:YOUR_KEY_SECRET' });

// 또는 2. 인스턴스 직접 생성
import { createHiggsfieldClient } from '@higgsfield/client/v2';
const client = createHiggsfieldClient({
  credentials: 'YOUR_KEY_ID:YOUR_KEY_SECRET',
  baseURL: 'https://platform.higgsfield.ai'
});


                            
3. Text-to-Image 생성 요청 (Subscribe)

                            
이미지 생성 태스크는 subscribe() 메소드에 모델 ID와 입력을 전달하여 실행하고, withPolling: true를 설정해 상태를 추적할 수 있습니다.

                            
const jobSet = await higgsfield.subscribe('flux_2/pro', {
  input: {
    prompt: 'A beautiful retro neon cityscape with flying cars, cyber punk, 8k',
    aspect_ratio: '16:9',
    safety_tolerance: 2,
    seed: 42
  },
  withPolling: true
});

if (jobSet.isCompleted) {
  const imageUrl = jobSet.jobs[0].results?.raw.url;
  console.log('생성 완료 이미지 URL:', imageUrl);
}


                            
4. Text-to-Video / Image-to-Video 생성

                            
비디오 생성 태스크 또한 동일한 패턴을 따르며, 비디오 모델 ID와 모션 파라미터를 추가로 제공합니다.

                            
// Text-to-Video 예시
const videoJob = await higgsfield.subscribe('veo-3-1-preview', {
  input: {
    prompt: 'Cinematic tracking shot of a white sports car driving down a coastal highway',
    aspect_ratio: '16:9',
    duration: 4, // 초 단위
    motion_strength: 7 // 1~10
  },
  withPolling: true
});

// Image-to-Video 예시 (기존 이미지를 소스로 활용)
const img2VideoJob = await higgsfield.subscribe('kling_omni_image', {
  input: {
    prompt: 'Gentle waves lapping on the sand, sunbeams dancing on water',
    image: 'https://example.com/source-ocean.jpg',
    aspect_ratio: '16:9',
    duration: 8,
    motion_strength: 5
  },
  withPolling: true
});


                            
5. API 상태 코드 일람 (Status Lifecycle)

                            

                                
                                    

                                        상태명 (Status)
                                        설명
                                    

                                
                                
                                    

                                        queued
                                        생성 대기열에 등록된 상태로, GPU 연동을 대기 중입니다.
                                    

                                    

                                        in_progress
                                        모델 가중치를 불러오고, 노이즈 제거 단계를 거쳐 실제 프레임을 생성 중입니다.
                                    

                                    

                                        completed
                                        생성이 안전하게 완료되어 결과물 미디어 다운로드 URL이 반환된 상태입니다.
                                    

                                    

                                        nsfw
                                        콘텐츠 안전성 가이드라인을 위배하여 거절되었습니다. (소모 크레딧은 전액 환불됩니다)
                                    

                                    

                                        failed
                                        시스템 또는 렌더링 에러가 발생했습니다. (소모 크레딧은 환불됩니다)
